# minishell_shl13_execute
execute version of minishell shl13 
 
 
## you can't do 'make'

## 실행방법 
```
./minishell 
```
을 터미널에 치기 

termcap도 적용되어 있어서 위 아래 오른쪽 왼쪽 키도 적용할 수 있습니다. 

cat, cd, pwd, ls 등 여러 명령어를 쳐보세요
